# TNBC-lung-metastases
Code for data analysis (Bedoya-López et al. Comms Bio, 2026). The newly generated epigenetic data from this study have been uploaded to the NCBI-GEO database under the code GSE305869. Each branch correspond to the analysis of each cohort (our XEN model, the AURORA US Metastasis Project, or external clinical validation with databases with lung metastasis events).
